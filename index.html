интернет магазин литературы

описание

Проект представляет собой магазин, оказывающий услуги по продаже литературных изданий. Любой покупатель может найти его через поисковую систему, выбрать товар из списка предложенных, добавить его в корзину и оформить покупку. При желании он может создать учетную запись, добавлять товары в избранное, просматривать историю покупок.

  • Широкий ассортимент продукции
  • Разработанная система скидок
  • Быстрая окупаемость проекта
  • Оперативная обработка клиентских запросов
